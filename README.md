## Sections of the Portfolio

The portfolio website consists of the following sections:

- **Home**: Introduction and a brief overview.
- **Skills**: A detailed list of my technical skills.
- **Experience**: My professional journey and work experience.
- **Education**: Academic background and certifications.
- **Projects**: A showcase of the projects I've worked on.
- **Contact**: Information on how to reach out to me.

---

## рҹ’» Technologies Used
- **Frontend:** React.js with Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Deployment:** Vercel

# Getting Started рҹҺҜ

### Fork and Clone the Repository рҹҡҖ
1. Click the **Fork** button at the top-right corner of the page to create your own copy of the repository.
2. After forking, open your terminal and run the following commands to clone the repo:

  ```bash
  git clone https://github.com/patowari/portfolio.git
  ```
Navigate to the Project Directory рҹ“Ӯ
Once the repository is cloned, change your directory to the project folder:
```bash
cd portfolio
```

Install Dependencies вҡҷпёҸ
From the root directory of your project, install the necessary packages:
```bash
npm install
```

Run the Development Server рҹҡҖ
Start the development server to see your project live:
```bash
npm run dev
```

View the Project рҹҢҗ
Open your browser and visit http://localhost:5173/ to see the result! рҹҺү
